# Rust + Axum + Fly.io Mini Project


## Running
`cargo run` to run locally
`fly deploy` to deploy to fly.io


## Database
`sqlx migrate run` 

# Features
[] DB w/ fly.io
[x] Hosted on fly.io
[] Texting (Twilio)

[] Route: Show transactions for category
[] Route: Show all transactions
